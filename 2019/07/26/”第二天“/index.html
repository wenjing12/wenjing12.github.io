我的手机 2019/7/26 14:08:46
mkdir 创建目录
ls显示目录中的文件
touch创建文件
rm删除文件 输入y（是） 回车（不是）
rm-f 强制删除文件（不询问是或者不是）
rm-r删除目录
cd 切换目录
命令加选项加参数
yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update检查是否需要更新
yum update APPNAME更新指定软件
yum update 更新所有软件
yum remove APPNAME卸载指定软件
