title: ”夏令营感悟“
date: 2019-07-26 15:16:39
tags:
—短暂的五天夏令营结束了，虽然短暂却很充实，回忆起来还是有很多值得细细品味的地方。
首先，感觉大家都特友善在去找寝室的时候，遇到了另外两个营员，然后和他们边走边聊，没有一点陌生人的感觉，就像多年未见的朋友一样，感觉特别亲切
一起吃饭，一起活动，分别时一起合照，期待下一次的相聚，心中担心与期许并存，担心的是我们也许以后再也见不了面了，期许的是我们明年可能将再次在相会。真的，他们都很优秀，所以期许比担心更多一点。
然后，我感受到了满满的正能量。在来夏令营的前几天，由于是暑假嘛，生活非常的堕落。到夏令营的第一天晚上，我从其他营员的寝室门口经过时，看到他们刚来就开始学习，着实震惊，也为自己最近的状态感到羞愧。杜老师讲课风趣幽默对待作业却又严谨，初步了解了Linux这款免费的操作系统，用户可以通过网络或其他途径免费获得，并可以任意修改其源代码。这是其他的操作系统所做不到的，深受同学们喜爱，短短几天的时间就让我们学到了很多干货比如学会了建一个个人的博客。
我和我的同学每天认真地对待每天的活动，满载一天的收获回到寝室。在寝室，和室友们畅所欲言，聊人生，聊梦想。在精力有剩余的时候再整理下知识，而疲倦的时候就早早休息，以良好的精神状态迎接之后的每一天。虽然夏令营的时间很短但却教会我们如何去学习一个东西，我想信对于我们来说夏令营的结束只是一个开始我们变得更努力更优秀的一个开始。同时也很感谢在学习同学老师对我的帮助我也希望我们能够一起变得更加优秀！

—hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
缩写：
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy
组合命令：
hexo s -g #生成并本地预览
hexo d -g #生成并上传

—Github账户注册和新建项目，项目必须要遵守格式：账户名.github.io，不然接下来会有很多麻烦。并且需要勾选Initialize this repository with a README
在建好的项目右侧有个settings按钮，点击它，向下拉到GitHub Pages，你会看到那边有个网址，访问它，你将会惊奇的发现该项目已经被部署到网络上，能够通过外网来访问它。安装Hexo，在自己认为合适的地方创个文件夹，我是在D盘建了一个blog文件夹。然后通过命令行进入到该文件夹里面
输入npm install hexo -g，开始安装Hexo
看到后面的“Start blogging with Hexo！”，激动有木有！！！！！
输入npm install，安装所需要的组件
输入hexo g，首次体验Hexo
